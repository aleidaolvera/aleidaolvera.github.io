What you will learn


HTML and CSS logo take from here.

This webpage is re-designed because of the TSA Webmaster requirements for 2014. The website should be separate from the TSA Chapter & CTE Webpages.

In these tutorials, you will learn about HTML and CSS languages that make designing and creating a website possible. HTML is an acronym for Hyper Text Markup Language. HTML is basically the language that allows a Webpage to be created. HTML focuses on adding content and minimal style. You can add text to websites by using HTML and can also edit fonts, colors, and add pictures or graphics. CSS is an acronym for Cascading Style Sheets. CSS a feature used in HTML to allow control over the style of your website content. Style Sheets are used to create a different appearance for items such as headers and links.

At the end of these tutorials you will be able to create this very website! This design brief webpage was designed with minimal HTML and CSS to show you an example of what you can do with very basic knowledge of HTML and CSS.

The tutorials are all divided into lessons and can all be found in our Tutorial page.
