What is CSS?

CSS is an acronym for Cascading Style Sheets. CSS a feature used in HTML to allow control over the style of your website content. Style Sheets are used to create a different appearance for items such as headers and links. Below is an example of what a CSS style would look like:

The basic point of CSS is to give style to text and different elements that were already established in the HTML of a website. It determines the output of content and the overall look of a webpage. CSS is what brings your HTML and website to life.
