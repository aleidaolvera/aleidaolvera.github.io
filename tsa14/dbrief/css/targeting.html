Targeting, Classes, IDs, Divs

Remember that HTML lesson about Classes, IDs, and DIVs (divisions)? Well now we are going to put that into play and explain what those names mean and how to target them.

Target Selector: URLs with an # followed by an anchor name, link to a certain element within a document. The element being linked to is the target element. The :target selector can be used to style the current active target element.

Classes- HTML elements with classes in them have specific styles. Class styles can be used over and over again. To target a class and give it the specific styles you want just put a (.) and the name you would like for your class.

IDs- IDs are usually meant for one element, just like people IDs belong to one specific element. IDs should not be used for multiple elements; it is allowed in CSS, but it is not recommended. To style a specific element with a unique ID you must use the (#) pound sign followed by the ID name.

These are the most essential tools to a webpage, you are almost getting to the point where you will be ready to create your own website!
