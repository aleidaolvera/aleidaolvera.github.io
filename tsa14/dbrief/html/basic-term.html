Basics and Terminology

HTML is a language for describing web pages. HTML stands for Hyper Text Markup Language. HTML is a markup language which means it is made up of markup tags that depict a document’s content. HTML documents contain HTML tags and plain text which are embedded within the tags. HTML document is essentially another name for web page.

Common Terms in HTML:

  • Tags- are made up of a keyword or term that are inside the angle brackets ( < > ). Information can go between two tags and the closing one is prefixed with a slash < / keyterm > Tags can also display attributes for example class, id, and href.
  • Attributes- Attributes are used to give more information about an element. Attributes are always displayed in the start tag. Attributes come in name/value pairs like: name="value" and are always enclosed in quotations.
  • Element- An HTML element starts with a start tag and ends with an closing tag. The element content is everything between the start and the end tag. Most HTML elements can have attributes.

In HTML, there is a basic structure to follow. It’s almost like a skeleton that all HTML documents follow. It starts with < !DOCTYPE html >. It is important to remember and stick by the basic HTML skeleton because it is the only way the HTML will be read properly.
